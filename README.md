# CCC solutions

My solutions to the **Canadian Computing Competition** (CCC) run by the University of Waterloo. Solutions are mostly in Python.

It's really messy and there are probably a lot of files with solutions that straight up don't pass on DMOJ (online judge).

[my DMOJ account 😭](https://dmoj.ca/user/henhen)

---

## Organization

- Solutions are grouped into folders by contest year
- Each file is prefixed by the difficultly/contest of the year (e.g. `j5_harvestwaterloo.py`)
- **`uva/`** — extra practice problems from the UVa Online Judge, named by problem ID
---

## File naming

Within each year, files are named by division and problem number:

- **`j1` … `j5`** — **Junior** division problems
- **`s1` … `s5`** — **Senior** division problems


Some folders also include:

- **`*input.txt` / `input.txt` / `output.txt`** — sample test cases for a problem
- **`main`** — a compiled binary
